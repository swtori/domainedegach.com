# Médias du site public

Ce dépôt inclut des **images SVG de remplacement** pour garantir un clone fonctionnel sans binaires lourds.

## Remplacer par vos médias

| Emplacement | Usage |
|-------------|--------|
| `img/img_gach/logo.svg` | Favicon et logo |
| `img/img_gach/backgroundGach.svg` | Fond des pages hero |
| `img/img_background/arbreGach.svg` | Fond secondaire |
| `img/img_room/_miniatures/*.svg` | Miniatures chambres |
| `img/placeholder.svg` | Carrousels et visuels « Autour » |

Adaptez les chemins dans les fichiers HTML/CSS si vous réintroduisez des `.jpg` / `.webp` d'origine.
